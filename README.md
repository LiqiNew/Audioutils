# Audioutils
Audioutils-音频录制和音频播放工具<br>  
音频录制有两个对象：
===
<br> 
MediaRecorderUtils对象是针对com.kailashdabhi:om-recorder框架封装，并且实现转换成wav格式。<br> 
MediaRecorderUtilsAmr对象是针对原生API封装，并且实现转换成amr格式<br> 
音频播放对象：
===
<br> 
MediaPlayerUtils对象是针对原生API封装的。
